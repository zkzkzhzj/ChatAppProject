# ChatAppProject
(가제) 마음의 고향 프로젝트

## 계속해서 개선해야할 사항
1. 아키텍처 및 ERD 최적화
2. 채팅 서비스, 실시간 채팅의 기술적 이슈 리서치
3. 공간에 대한 구현 방향 고민
4. 윤리적인 문제 해결

### 프로젝트 방향 기록
힘든 마음을 가지고 있는 사람이 편하게 마음을 털어놓고 대화로 고향에 온 듯 느낌을 받을 수 있는 곳

문제점)
- 대화를 할 사람들을 처음에 어떻게 모집할 것인가?
- 카카오톡 익명채팅/블라인드를 사용하지 않고 이 앱을 사용할 이유가 무엇인가?
- 수익모델은 어떻게 할 것인가?
- 윤리적인 문제는 어떻게 할 것인가?(대화의 퀄리티, 비난)

방향성 수정)
- 고향을 키 포인트로 잡고 공간을 활용하자
- AI를 활용한 채팅 대화 기능 지원
- 내 공간 꾸미기를 활용하여 소속감 부여

장소를 메인으로 잡은 의사소통 서비스
- 고향에 온듯한 편안함과 꾸미기의 요소 활용
- AI 상담은 중심이 아닌 보조 수단
- 텍스트 채팅 서비스를 넘어서 마이크, 화면 공유까지 생각
