# Branch Cleanup

## 목적

머지 또는 중단된 작업 후 로컬/원격 브랜치 정리를 안전하게 수행한다.

## 절차

1. 현재 브랜치와 작업 트리 상태를 확인한다.
2. 삭제 대상 브랜치가 머지되었거나 명시적으로 중단되었는지 확인한다.
3. 사용자 변경을 포함한 브랜치를 임의로 삭제하지 않는다.
4. 원격 삭제는 사용자 요청이 있을 때만 수행한다.
