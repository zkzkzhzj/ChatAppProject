# Wiki Lint

## 목적

`docs/wiki/` 문서가 인덱스와 정책을 따르는지 확인한다.

## 절차

1. `docs/wiki/INDEX.md` 링크가 유효한지 확인한다.
2. 새 wiki 문서가 인덱스에 등록되어 있는지 확인한다.
3. 코드 변경과 wiki 내용이 충돌하지 않는지 확인한다.
4. wiki 변경이 필요한데 빠졌다면 보고한다.
