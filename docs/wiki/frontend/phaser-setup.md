# Phaser.js + Next.js 통합 기록

이 문서는 과거 Phaser 2D 프로토타입 시절의 기록이다.
현재 프론트엔드 런타임은 `frontend/src/three/`의 Three.js scene 구조를 사용하며, 이 페이지를 현재 아키텍처 기준으로 참조하지 않는다.

Phaser 기반 구조와 `PhaserCanvas`, `PixelArtScene` 같은 명칭은 현재 코드의 활성 구조가 아니다.
과거 전환 배경은 `docs/learning/72-phaser-to-threejs-pivot-decision.md`를 참고한다.
